============================================
  codex-switch —— 给小白的使用手册
============================================

这个程序可以让你的 OpenAI Codex App 改用 DeepSeek、Kimi、智谱、通义
等任意 OpenAI 兼容的国产/海外大模型，一键切换，不用换 Codex 客户端。

============================================
【准备一次性工作】

1. 装好 OpenAI Codex App（已经在用就行）
2. 去你要用的模型平台注册账号、充值、复制 API Key：
     - DeepSeek:  https://platform.deepseek.com
     - Kimi:      https://platform.kimi.com
     - 智谱 GLM:  https://open.bigmodel.cn
     - 阿里通义:  https://dashscope.console.aliyun.com
     - 其他自查
   每家的 Key 都长这样：sk- 开头的一长串字符

============================================
【启动用法（3 步）】

第 1 步：双击 codex-switch.exe
   - 第一次启动可能慢 5-10 秒（正常，在解压程序）
   - 如果 Windows 红字警告"未识别"→ 点"更多信息"→"仍要运行"
   - 如果 Defender 拦 → 文件右键 → 属性 → 勾"解除锁定"→ 确定

第 2 步：在窗口里
   - 模型下拉里选一个（默认列出 DeepSeek×2、Kimi×4）
   - 把对应平台的 API Key 粘到 Key 输入框
   - 点蓝色大按钮【▶ 启动翻译官 + 切到所选模型】
   - 日志区写"全部就绪"就行了

第 3 步：打开 Codex App
   - Codex 现在已经在用你选的模型
   - Codex 输入框右下角会显示当前模型名（如 deepseek-v4-pro）
   - 重要：切换器窗口不要关，关了就连不上翻译官

============================================
【加更多模型（自定义提供商）】

主界面下拉旁边有个【+ 自定义】按钮，点开：

  - 顶部"模板"下拉：内置 9 个常用平台模板（智谱/通义/零一/豆包/
    Groq/Mistral/OpenRouter 等），选了自动填好 Base URL
  - 5 个字段（显示名/模型 ID/Base URL/Key 入口/API Key）
  - 填好点保存 → 主下拉就多一个带 ✦ 标记的选项

不在模板里的模型？模板选"自定义"，4 个字段全手填：
  - 显示名：自己起，例如"我的代理"
  - 模型 ID：去那家文档查（如 glm-4-plus / qwen-max）
  - Base URL：去那家文档查 chat/completions 完整端点
  - API Key：那家给你的 sk- 开头字符串

【管理】按钮可以列出所有自定义条目、删掉不要的。

============================================
【不用了，想切回 OpenAI】

   - 切换器窗口里点红色按钮【⏸ 停止翻译官 + 切回 OpenAI】
   - 重启 Codex App
   - 完事

============================================
【日常使用】

   - 每次想用别的模型：双击 .exe → 选模型 → 蓝色按钮
   - 想切回 OpenAI：红色按钮 → 关窗口
   - Key 会按厂商分开记住，下次自动加载，不用重输

============================================
【出问题怎么办】

▸ 双击没反应
  → 等 10 秒，第一次启动慢
  → 还不行：右键 → 属性 → 勾"解除锁定"

▸ Codex 报连接失败
  → 切换器窗口关了，重新打开它

▸ Codex 报 401 / Invalid API Key
  → 该平台的 API Key 错了或余额没了，去对应平台查

▸ Codex 报 unknown model / 404
  → 自定义条目里的"模型 ID"写错了，去那家文档对一下
  → 或者 Base URL 不对，确认末尾是 /chat/completions

▸ Codex 还显示 OpenAI / GPT
  → 彻底退出 Codex（任务栏右键退出）再重开

▸ 想彻底卸载
  → 删 codex-switch.exe
  → 切换器之前帮你备份了 Codex 原配置在：
    C:\Users\<你的用户名>\.codex\config.toml.openai-backup
  → 把这个 .openai-backup 文件复制为 config.toml（覆盖）就回到原状

============================================
【安全说明】

  - 所有 API Key 只保存在你本机这两个文件里：
    C:\Users\<你的用户名>\.cc-switch\switcher-keys.json
    C:\Users\<你的用户名>\.cc-switch\switcher-custom-providers.json
  - 翻译官只在本机 127.0.0.1:18667 监听，外网访问不到
  - 切换器第一次会自动备份你原 Codex 配置，不会覆盖丢失
  - API Key 只发往你选的那家上游 API（如 api.deepseek.com），
    不发往任何第三方
